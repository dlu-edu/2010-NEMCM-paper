\section{问题重述}
\frame{\frametitle{封闭系统货币交易问题}
\begin{itemize}
\item \textbf{封闭系统}:$N$个人构成,每人有$m$单位的货币量.
\item \textbf{货币守恒}:在整个过程中总的货币量保持不变．
\item \textbf{交易规则}:货币通过交易实现转移.试给出两种交易规则.
\item \textbf{货币分布}:每个规则下，进行充分长时间后的货币分布情况.
\end{itemize}

}

\frame[<+->]{
  \frametitle{具体规则}

已有的货币转移规则：

 \begin{itemize}
 \item \textbf{规则一}：每次交易过程中，任意选出两个交易者 $i$ 和 $j$，将
他们之间的货币交易量记为 $\Delta S$，则交易量为
\[
\Delta S = \frac{1}{2}\varepsilon(m_i+m_j)
\]
 \item \textbf{规则二}：限制交易者的货币量足够交易时才能交易.
 \item \textbf{规则三}：有保存进行交易存储率。存储率分为定值 ($s=s_i=s_j$) 和随机值两种．
  \end{itemize}
 }

\frame[<+->]{
  \frametitle{具体规则}
  对已有的货币转移规则的改进：

 \begin{itemize}
 \item \textbf{规则四}：每次参与的交易者大于等于两个，假设某次交易有$k$个交易者，因此参照已有规则．我们有以下交易规则：
\[
m_i(t+1)=m_i(t)+\varepsilon\Big(\sum_k^{i=1}m_i(t)\Big)
\]
其中$\varepsilon_i=\xi_i-\frac{1}{k}\sum_{i=1}^k\xi_i$，$\xi_i$
为[0,1]间随机正数．
 \end{itemize}
 }

\section{元胞自动机模型}

\frame[<+->]{
  \frametitle{元胞自动机模型}
  由于地理以及其它条件的限制和约束，交换只发生在局部．考虑到模型一的这种缺陷，本文建立了元胞自动机模型.

  \begin{center}
  \includegraphics[width=0.4\textwidth]{./figure/fig10.pdf}　
  \end{center}
}

\frame[<+->]{
  \frametitle{元胞自动机模型}
假设该人和它的某个邻
居的货币量分别为$m_{ij}$和$m_{ij+1}$，则它们间的货币交易量和交易规则可描述为

\[\Delta S=\frac{1}{2}\varepsilon\Big(\xi>p_{ij}\Big)\Big(m_{ij}+m_{ij+1}\Big)
\]

\[m_{ij}(t+1)=m_{ij}(t)+\Delta S
\]

\[m_{ij+1}(t+1)=m_{ij+1}(t)-\Delta S
\]

其中 $\xi$ 是区间 $[0,1]$ 中的随机数，如果 $\xi>p_{ij}$ 成立， 则
$(\xi>p_{ij})=1$, 否则为０ }

\section{结论及模型优缺点}

\frame{ \frametitle{结论} 本文成功地解决了封闭系统内的货币转移问题:
\begin{itemize}
\item 模型一中的改进规则四，对研究货币转移速度提供了借鉴依据;
\item 模型二即元胞自动机模型则为研究局部区域出现的贫富差距提供了新的思路
\end{itemize}
}

\frame{ \frametitle{模型优缺点}
\begin{itemize}
\item \textbf{优点}：本文不仅考虑已有的三种交易规则，还提出了第四种规则：一次交
易中有多人参与交易的情况，并且得到了各个规则的货币量分布图。同
时讨论了导致各个结果之间差异的原因。本文提出的元胞自动机模型，把实际中交易的局限性也很好的考虑进去，
并且还发现交易概率的差异会导致局部贫穷或富裕。
\item \textbf{优点}：结果都是宏观的，可能多少会忽略某个人或某个群体在交易过程中的变化情况。不
过在本文的模型中，这种某个人或某个群体在交易过程中的变化情况是可以得到的。
本文的元胞自动机模型，忽略了人口的流动性和主观能动性，这在一定程度上使得模型相对比较局限。

\end{itemize}
}
